# Anna Magic
Простой сайт-визитка для таролога Анны.

## Порядок установки
1. Убедиться в доступности в вашей системе `npm`.
2. Клонировать репозиторий.
```Bash
git clone https://github.com/WorldTradeCom/anna-magic
cd anna-magic
```
3. Установить зависимости.
```Bash
npm install
```
4. Собрать оптимизированную версию.
```Bash
npm run build
```
5. Запустить релизную версию на требуемом порте.
```Bash
npm run start -p {PORT}
```
